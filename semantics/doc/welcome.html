Welcome to the JADE Semantics Add-on


The JADE Semantics Add-on is a framework that eases the development of cognitive agents with automatic understanding capabilities of FIPA-ACL messages. It also provides developers with a high-level programming style, based on the abstract concepts of the FIPA-SL logical language (beliefs, intentions, facts expressed as logical formulas, action expressions, ...).

It is developed and made available under the terms of the LGPL open source license by the Research & Development division of the France Telecom Group.

To get started with developing with this add-on, you can follow the tutorial. It consists in a detailed guided tour of the framework together with some progressive practical exercises.

Experienced JSA developers should also consult the guide for migrating their applications from the JSA 1.2 version to the JSA 1.4 version.

From the version 1.4, we are also starting a “cook-book” documentation, which will also be available and progressively enriched on-line. It aims at helping developers with some practical code examples, which they can easily adapt.

The API reference can be found here (note you need to compile the Javadoc first from your distribution by launching the ant doc command in the installation directory).

Previous documentation (version 1.2) are available in this directory.
